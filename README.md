# Quick Setup
## Blockchain setup
install [Truffle](https://www.trufflesuite.com/docs/truffle/getting-started/installation) and run:
```
truffle develop
```
then deploy contracts with command in console:
```
migrate --reset
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
